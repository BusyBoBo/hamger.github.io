实现双向绑定

发表于 2017-09-24 |

Vue 的一个最明显的特性就是其不太引人注意的响应式系统。数据模型仅仅是普通的 JavaScript 对象。而当你修改它们时，视图会进行更新。
这是如何做到的呢？下面我将实现一个双向绑定的简单案例，在实现案例后我相信你对 Vue 响应式系统的原理一定会有所了解。我将分三步来实现：

  1. model -> view 初始化
  2. view -> model 绑定
  3. model -> view 绑定
